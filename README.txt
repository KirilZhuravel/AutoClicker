  ЗАПУСК
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  Дважды кликни на autoclicker.exe
  Появится небольшое окно — можно работать.

  ! Если Windows SmartScreen блокирует запуск:
    → Нажми "Подробнее" → "Всё равно запустить"
    Это нормально для любого .exe без подписи.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  РЕЖИМ МЫШЬ
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  1. Выбери радио-кнопку "Мышь" (выбрана по умолчанию)

  2. Кнопка — какую кнопку мыши нажимать:
       Left   — левая (обычный клик)
       Right  — правая (контекстное меню)
       Middle — средняя (колёсико)

  3. Тип — Single (одиночный) или Double (двойной клик)

  4. Позиция — в окне отображаются текущие координаты
     курсора в реальном времени (X / Y на экране).
     Кликер будет кликать ТУДА, где курсор находится
     в момент каждого тика.

  5. Фиксировать курсор — если поставить галочку,
     позиция запоминается при нажатии Старт и курсор
     возвращается на это место после каждого клика.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  РЕЖИМ КЛАВИАТУРА
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  1. Выбери радио-кнопку "Клавиатура"

  2. Нажми кнопку "Записать"

  3. Нажми нужную клавишу на клавиатуре —
     она запомнится и отобразится в окне.
     (Escape отменяет запись, F3 зарезервирован)

  4. Клик идёт в АКТИВНОЕ окно на экране —
     то есть в то приложение, которое сейчас
     в фокусе. Переключись на нужное окно перед
     запуском или используй F3 не трогая мышь.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  ИНТЕРВАЛ
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  Секунды + Мс = суммарная пауза между кликами.

  Примеры:
    Сек: 0 / Мс: 100  →  10 кликов в секунду
    Сек: 0 / Мс: 50   →  20 кликов в секунду
    Сек: 1 / Мс: 0    →   1 клик  в секунду
    Сек: 0 / Мс: 1    → ~1000 кликов в секунду (!)

  ! Интервал меньше 10 мс может работать
    нестабильно — зависит от системы.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  СТАРТ / СТОП
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  Кнопка "Старт [F3]" в окне программы
  — ИЛИ —
  Клавиша F3 с ЛЮБОГО окна (глобальный хоткей).

  Счётчик "Кликов" сбрасывается при каждом старте.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  ТИПИЧНЫЙ СЦЕНАРИЙ ИСПОЛЬЗОВАНИЯ
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  Кликер по кнопке в браузере:
    1. Режим: Мышь, Left, Single
    2. Интервал: 0 сек / 200 мс
    3. Наведи курсор на нужную кнопку в браузере
    4. Нажми F3 — кликер начал работу
    5. Не двигай мышь (или включи "Фиксировать курсор")
    6. F3 — остановить

  Автоматический ввод клавиши (например, E в игре):
    1. Режим: Клавиатура
    2. Записать → нажать E
    3. Интервал: 0 сек / 500 мс
    4. Переключись на игровое окно
    5. F3 — старт, F3 — стоп
